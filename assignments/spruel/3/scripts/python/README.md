# Overview

For this assignment I ended up creating a Python Playwright to collect the cookies from my assigned URLs.
Playwright is a tool that allows you to automate controlling a web browsers. It is typically used to automate the testing of websites and web apps.
This option was selected for this project to address http/2, capcha, and other issue I ran into when trying to use curl to collect the cookies.
The script takes an input file with the URL of sites that cookies will be collected from, and the input file should be placed in the same directory of the script.
Note: Some sice may be unreachable or cookie counts may differ from run to run depending on security measures setup on the various sites. 
The script can be found in the scripts/python folder:
[scripts/python](scripts/python)
-----------------

The original c++ script using libcurl can be found in the scripts/c++ folder:
[scripts/c++](scripts/c++)
